Раздел блог на сайте Distillery.com

Цель:

к уже существующему статическому сайту добавить раздел "Блог" на базе CMS WordPress. Страницы раздела блога должный полностью соответствовать фирменному стилю и не отличатся от других страниц сайта, даже в случае изменения дизайна основной части сайта. Сайт должен поддерживаться всеми мобильными устройствами и платформами, всеми браузерами последних версий, включая IE11.

Что делалось:

  • разработка функциональности блога с использованием Worpress API (категории, метки, поиск и т.д.),
  • верстка страниц по дизайн макетам в 3-х разрешениях,
  • разработка функциональности для автоматического кадрирования и изменения размера пользовательского контента,
  • настройка https протокола,
  • подключение и настройка модуля push-уведомлений,

Использованные технологии:

  • Wordpress API
  • CSS/HTML
  • JavaScript
  • JQuery
  • SVG
  • Open Graph

Результаты и достижения:

Полностью готовый и протестированный раздел "Блог" с адаптивностью под любые устройства. Корректная обработка пользовательского контента в соответствии с дизайн-макетами, деградация функциональности на старых браузерах, интеграция стилей со статическим сайтом, работа в большой территориально распределенной команде через Skype, таск-трекер Jira и настроенный Git-flow.

Дата запуска: октябрь 2016
