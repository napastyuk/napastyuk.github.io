Что делалось:

  • конвертация всей мелкой графики в SVG
  • оптимизация под экраны Retina
  • подсвечивание элементов разворота журнала через SVG изображение
  • создание интерактивного блока "печатный станок" JQuery
  • использование JQuery UI slider для управления проигрыванием видео
  • использование JQuery UI Draggable Object для реализации прокрутки экрана телефона в блоке "телефон в руке"
  • добавление логотипа компании на интерактивную карту Яндекс

Использованные технологии:

  • HTML 5
  • CSS 3
  • SVG
  • JavaScript
  • JQuery
  • JQuery UI
  • Yandex Map API

Результаты и достижения:

Освоил создание интерактивных svg изображений, более детально изучил использование JavaScript/JQuery для создания интерактивных элементов. При работе над проектом активно использовалась система контроля версий GIT.
