Лэндинг Корпоравная пресса - Десктопная версия

Цель:

разработать интерактивный лендинг про услугу "создание корпоративной прессы"

Что делалось:

  • так как дизайн-макет был в формате Illustrator, всю мелкую графику перевел в svg
  • для оптимизации под экраны Retina большая часть фотографий на сервере хранится в разрешении x4
  • для блока "разворот журнала в деталях" в SVG файл было добавленно подсвечивание блоков через CSS hover
  • для блока "принт-менеджмент" был встроен плеер, который играет зацикленно MP4 видео c записью печати газеты. Для управления проигрыванием добавлен компонент JQuery UI slider. Формат MP4 показал наименьший размер для данного анимированного изображения.
  • Для блока "телефон в руке" с прокручивающимся экраном, был использован компонент JQuery UI Draggable Object
  • Для добавления логотипа на карту Яндекс, использовался API Яндекс.Карт
  • Все остальные эффекты сделаны с помощью JQuery функций или плагинов

Использованные технологии:

  • HTML 5
  • CSS 3
  • SVG
  • JavaScript
  • JQuery
  • JQuery UI
  • Yandex Map API

Результаты и достижения:

Освоил создание интерактивных svg изобажений, более детально изучил использование JavaScript/JQuery для создания интерактивных элементов. При работе над проектом активно использовалась система контроля версий GIT.

Дата запуска: Апрель 2016
